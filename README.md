# 使用说明

一键搭建脚本 (环境选择`Bash` 或 `Blank Repl`)   

Go-File-Replit一键脚本
```
bash <(curl -s https://raw.githubusercontent.com/sxbai/go-file-replit/master/install.sh)
```
# 关于后续更新
